== Description ==\
\
\pard\pardeftab720
\cf0 Youtube Videos Widget lets you add the latest youtube video anywhere in your Wordpress blog. You can add it to a post or a page with the short code [youtube_videos], or to a sidebar with the included widget. Customize options in the options page and on the widget itself.\

\cf0 \
== Installation ==\
\
This section describes how to install the plugin and get it working.\
\
e.g.\
\
1. Upload `youtube-videos.php` to the `/wp-content/plugins/` directory\
2. Activate the plugin through the 'Plugins' menu in WordPress\
3. Place `<?php do_shortcode('youtube_videos'); ?>` in your templates where you want your video to appear or place the widget in any sidebar.\
\
== Frequently Asked Questions ==\
\
